# 我的在线简历

嵌入式软件工程师简历，通过 GitHub Pages 在线托管，内容与样式分离。

## 在线访问

https://tddyyds.github.io/resume/

## 文件说明

- `index.html` —— 样式和排版模板，**不用改**
- `data.js` —— 简历内容（文字），**只改这个**
- `README.md` —— 本说明

## 如何在任意电脑上更新内容

1. 浏览器登录 github.com，打开本仓库的 `data.js`
2. 点右上角铅笔图标 ✏️ 编辑，只改引号 `" "` 里的文字（不要动 HTML 标签、逗号别漏）
3. 拉到底点 **Commit changes** 提交
4. 约 1 分钟后，访问网址自动更新

## 导出 PDF

打开网址 → 点右上角「⬇️ 导出 PDF」→ 打印窗口里**取消勾选「页眉页脚」**、**勾上「背景图形」** → 另存为 PDF。

## 上传照片

点右上角照片区域上传本地证件照（保存在当前浏览器，换设备需重新上传）。
